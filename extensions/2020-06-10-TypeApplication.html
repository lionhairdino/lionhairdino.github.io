확장 TypeApplications

Posted on June 10, 2020 
some :: Num n => n -> n
some x = x + x

> :set -XTypeApplications
> some @Int 1
2
> some @Double 1
2.0

some 함수가 n 컨텍스트에 있는데, n 을 임시로 특정 타입 컨텍스트로 실행하고 싶을 때 @를 붙여, 폴리모픽 함수를 특정 컨텍스트에서 실행할 수 있습니다.
